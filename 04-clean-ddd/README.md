# :information_source: Como Executar?

> **1.0.** Clone o Repositório:

```bash
git clone https://github.com/tavareshenrique/ignite-nodejs.git
```

> **1.1.** Acesse a pasta:

```bash
cd 04-clean-ddd
```

> **1.1.** Instale as dependências:

```bash
pnpm install
```
